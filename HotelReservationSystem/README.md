## Hotel Reservation System

This project is a simple Java GUI-based application used to manage hotel room bookings.

### Features:
- Book room with name, room number, and type
- Cancel booking
- Display bookings in table
- Simple and user-friendly interface

### Technologies Used:
- Java
- Swing (GUI)
- ArrayList

### Description:
This system allows users to book and cancel rooms. Data is stored using ArrayList and displayed in JTable. It helps in understanding GUI design and event handling in Java.
